『引』前端知识总结

By 七点的沐兮

2016-07-08

从事前端工作已经两年了，这两年接触了不少技术。作为一个非科班出生的前端，在这条路上走无疑是需要在自学的过程中不断的总结和积累的。

最近看了@张鑫旭的前端微博，他专注前端，对前端知识的总结也比较全面。对于知识体系庞大的前端，在他的微博偶尔翻看并不能完全掌握，俗话说：“好记性不如烂笔头”。现在虽然笔头用得少了，但是我还是决定自己把看过的前端知识总结一下，以便加深理解。而且在有些常用的知识点上配上自己的demo,如果有幸得同行关注，或许对像我这些刚入选的菜鸟也可有些许帮助吧。

文章声明： 时间有限，为了尽快上手，看了不少前端同行的博客。个人比较欣赏@晚晴幽草轩和@帕兰映像的博客风格,暂且借用一下，特此声明，非常感谢二位。

更多良心分类清单，请参考：JS前端开发群规 - 492107297；其 Github 在线地址为 http://t.cn/RL2NtqX。这个前端分享学习QQ群氛围大好，十分难得。非常感谢群主 @豪情 提供这个交流平台。JS前端开发群规 - 492107297：这寄存于 看云，分类清晰，也在持续更新中。
